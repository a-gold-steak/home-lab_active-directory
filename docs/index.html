Introduction

This part of the project gives a brief overview on what was completed on this project and how to create a VM.

Windows Server 2022 Setup

In this section, I go over installing Windows Server 2022, setting up the network, creating a domain, creating an admin account, installing RAS and NAT, DHCP setup, a PowerShell script, and adding a Windows 10 Client to the domain.

Added an Ubuntu Client to the Domain

This section demonstrates how to add an Ubuntu client to the domain. There was two Linux clients added to the domain. One is an Ubuntu client, the other is a Kali client for pentesting purposes.

Installed Samba File Server

Here, I installed a Samba file server for file sharing. The Samba file server was hosted on the DC. Note, It is recommended to install a file server on a seperate server for security and performance purposes. This is just a demonstration.

Deployed Software to Clients with a GPO

In this section, I deployed software to Windows clients. In this demonstration, a Google Chrome .msi file was deployed.

Enabled SSH of Devices

This section demonstrates how to enable SSH on all the clients. SSH can be used to access a device remotely and securely.

Access Controls and Other Additions

In this part of the project, access controls were implemented as well as other minor additions that would make a sys admin's life easier. Some of these additions are blocking an application from use, file system quotas, and disabling the registry.

Deployed Folder Redirection for Windows Clients

This section demonstrates how to implement folder redirection for Windows clients. I installed the server to host the users files on the DC. Again, it is recommended to be installed on a seperate server.

Deployed Network-Wide Print Server

In this section, I installed a network-wide print server. The printer used was a network-only printer which did make implementation rather difficult, regardless, the printer was successfully deployed to the network.
